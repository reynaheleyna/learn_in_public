## Prompt Engineering 
- Craft clear instructions
- Ensure code is functionally syntactically and contextually correct 
- Generate context-appropriate code for project needs. 

## Principles and Best Practices of Prompt Engineering
1. Focus on a single well-defined task for clarity
2. Provide explicit and detailed instructions
3. Keep prompts concise to maintain clarity without overloading the AI 
4. Use descriptive filenames and keep related files open for context. 
5. Refine prompts based on feedback to improve AI responses
6. Use layered prompts: break complex tasks into smaller manageable prompts
7. Feedback loop: Continuously evaluate and adjust prompts for optimal performance
8. Collaborative Input: involve team members to ensure comprehensive and effective prompts.
